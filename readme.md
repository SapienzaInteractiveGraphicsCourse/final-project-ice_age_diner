# Ice Age Diner - Final Project

Questo è il progetto finale per il corso di **Interactive Graphics** presso la Sapienza Università di Roma. 
Il gioco è un ristorante 3D interattivo ispirato al classico "Penguin Dinner", sviluppato interamente in linguaggio JavaScript utilizzando la libreria grafica **Three.js**.
